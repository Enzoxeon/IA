# 🤖 Introducción a la Inteligencia Artificial y Machine Learning

## ¿Qué es la Inteligencia Artificial (IA)?

La **Inteligencia Artificial (IA)** es un campo de la informática que busca crear sistemas o programas capaces de realizar tareas que normalmente requieren **inteligencia humana**.

Estas tareas pueden incluir:

- 🧠 Aprender de datos  
- 🗣️ Entender lenguaje humano  
- 👁️ Reconocer imágenes  
- 📊 Tomar decisiones o hacer predicciones  

### Algunas áreas importantes dentro de la IA

- Machine Learning  
- Deep Learning  
- Natural Language Processing (NLP)  
- Computer Vision  

### Ejemplos de IA

- Un sistema que reconoce **caras en fotos** 📸  
- Un asistente virtual que entiende lo que dices 🗣️  
- Sistemas de recomendación como los de streaming 🎬  

---

# 📚 ¿Qué es Machine Learning (ML)?

**Machine Learning (Aprendizaje Automático)** es una rama de la IA en la que las computadoras **aprenden a partir de datos** en lugar de ser programadas con reglas explícitas.

En lugar de decirle al programa exactamente **qué hacer paso a paso**, se le dan **datos para que descubra patrones por sí mismo**.

### Ejemplo simple

Supongamos que queremos detectar **spam en emails**:

1. Le damos al sistema miles de emails.
2. Algunos están etiquetados como **spam**.
3. Otros están etiquetados como **no spam**.
4. El modelo aprende patrones.

Luego puede **clasificar nuevos emails automáticamente**.

---

# 🧩 Tipos de Machine Learning

Existen varios tipos de aprendizaje en Machine Learning.

---

## 1️⃣ Aprendizaje Supervisado (Supervised Learning)

En este tipo de aprendizaje, los datos **ya tienen la respuesta correcta**.

### Ejemplo

| Entrada | Respuesta |
|-------|--------|
| foto de gato | gato |
| foto de perro | perro |

El modelo aprende a **predecir la respuesta correcta**.

### Usos comunes

- 📧 Detección de spam  
- 🏥 Diagnóstico médico  
- 💰 Predicción de precios  
- 📈 Predicción de ventas  

### Tipos de problemas

**Clasificación**

Predecir una categoría.

Ejemplo:
- spam / no spam  
- gato / perro  

**Regresión**

Predecir un número.

Ejemplo:
- precio de una casa  
- temperatura  

---

## 2️⃣ Aprendizaje No Supervisado (Unsupervised Learning)

En este caso, los datos **no tienen etiquetas**.

El algoritmo intenta **encontrar patrones o grupos por sí solo**.

### Ejemplo

Un sistema analiza miles de clientes y los agrupa según:

- comportamiento de compra
- edad
- intereses

### Aplicaciones

- Segmentación de clientes  
- Detección de anomalías  
- Reducción de dimensionalidad  

---

## 3️⃣ Aprendizaje por Refuerzo (Reinforcement Learning)

En este tipo de aprendizaje, el modelo aprende **mediante prueba y error**.

El sistema recibe:

- 👍 **recompensas** cuando hace algo bien  
- 👎 **castigos** cuando hace algo mal  

Con el tiempo aprende a **tomar mejores decisiones**.

### Ejemplos

- Robots que aprenden a caminar 🤖  
- IA que aprende a jugar videojuegos 🎮  
- Sistemas de control automático  

---

# 📊 Resumen

| Tipo de Machine Learning | Cómo aprende | Ejemplo |
|---|---|---|
| Supervisado | Datos con respuestas | detectar spam |
| No supervisado | Buscar patrones | agrupar clientes |
| Refuerzo | Recompensa / castigo | videojuegos, robots |
